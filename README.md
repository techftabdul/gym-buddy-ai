# 🏋️ Gym Buddy AI

Gym Buddy is an AI-powered workout generator that creates a personalized 7-day gym routine based on a user's fitness goals.

## 🚀 Features

• Generate a complete 7-day workout plan
• AI-powered exercise recommendations
• Clean card-based UI
• Responsive design

## 🧠 AI Integration

Workout plans are generated using the Gemini API.

## 🛠 Tech Stack

React
Vite
Gemini API
JavaScript
CSS

## 📦 Installation

Clone the repository

git clone https://github.com/techftabdul/gym-buddy-ai

Install dependencies

npm install

Run development server

npm run dev
